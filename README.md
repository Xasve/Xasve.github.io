## About

This is my personal portfolio built as a single-page application. It highlights my skills, featured projects, and services as a freelance web developer available for remote work worldwide.

## Features

- Responsive design — works on mobile, tablet, and desktop
- Smooth scroll navigation with active section highlighting
- Animated skill progress bars and project cards
- Direct contact section with availability status
- Dark theme with modern typography

## Getting Started

No build tools or dependencies required. Just open `index.html` in any modern browser.

```bash
git clone https://github.com/Xasve/Xasve.github.io.git
cd Xasve.github.io
open index.html
```

## Deployment

This site is deployed automatically via **GitHub Pages** from the `main` branch. Any push to `main` updates the live site within a few seconds.
